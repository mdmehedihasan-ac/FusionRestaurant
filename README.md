# Fusion RSTRNT

Base frontend Vite + React + TypeScript per ricostruire e migliorare il sito di Fusion Restaurant.

## Stack

- Vite 7
- React 19
- TypeScript 5
- ESLint 9

## Stato attuale

- Workspace inizializzato nella root corrente.
- Template starter rimosso e sostituito con una shell editoriale piu adatta a un sito ristorante.
- Struttura pronta per essere popolata con contenuti, immagini e sezioni del sito originale.

## Comandi

- `npm run dev`
- `npm run build`
- `npm run lint`
